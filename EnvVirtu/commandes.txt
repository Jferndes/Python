Permet d’isoler un projet dans son propre environnement, indépendant des modules installés en local

1. Créer un répertoire et se positionner dedans

$ mkdir projet_numpy

$ cd projet_numpy

2. Créer l’environnement virtuel

$ python -m venv venv

3. Activer l’environnement

Sous Windows :

$ venv\Scripts\activate 

Sous macOS / Linux :

$ source venv/bin/activate Quand il est activé, ton invite de commande affichera quelque chose comme (venv) au début.

4. Installer les dépendances (les modules)

$ pip install -r requirements.txt

Pour vérifier l’installation (ex avec numpy comme module)

python -m pip show numpy

python -c "import numpy as np; print(np.version)"

Désactiver l’environnement

$ deactivate

Relancer = activez
